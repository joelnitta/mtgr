# Get a list of Magic the Gathering sets

Print a dataframe of expansion set codes and names.

## Usage

``` r
mr_get_sets()
```

## Value

Dataframe

## Details

Only includes expansion sets that have already been released. For access
to complete set data, see scryr::scry_sets(). Not all sets have data
available on 17lands.

## Examples

``` r
mr_get_sets()
#> Error: 
```
