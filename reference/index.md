# Package index

## Cache

Functions for handling locally stored data

- [`mr_cache_dir()`](mr_cache_dir.md) : Check the location of the cache

## Downloading data

Functions that download data

- [`mr_download_17lands_file()`](mr_download_17lands_file.md) : Download
  a 17lands data file

## Getting data into R

Functions that fetch and load data into R

- [`mr_get_17lands_data()`](mr_get_17lands_data.md) : Get 17lands data
- [`mr_get_17lands_cols()`](mr_get_17lands_cols.md) : Get column names
  of a 17lands data file
- [`mr_get_sets()`](mr_get_sets.md) : Get a list of Magic the Gathering
  sets

## Arrow

Functions that facilitate working with the {arrow} package for
out-of-memory data

- [`mr_make_17lands_schema()`](mr_make_17lands_schema.md) : Create an
  arrow schema for 17lands data
- [`mr_open_17lands_arrow()`](mr_open_17lands_arrow.md) : Open a dataset
  in arrow format

## Datasets

Datasets included with this package, mostly metadata

- [`replay_cols`](replay_cols.md) : Column names and types in replay
  data
